# 🌍 Transparent Crowdfunding Platform

Welcome to the most transparent way to crowdfund projects on the blockchain! This project uses the Stacks blockchain to ensure donations are secure, trackable, and only paid out when milestones are met.

## ✨ Features

💸 Secure STX contributions to campaigns  
🔍 Track every contribution and payout on-chain  
🗳 Vote on milestone approvals or disputes  
🔄 Refund donors if campaigns fail  
📊 View campaign progress and analytics  
🚫 Prevent fraudulent or duplicate campaigns  

## 🛠 How It Works

**For Campaign Creators**  
- Create a campaign with a funding goal and milestones  
- Submit milestone proof (e.g., IPFS hash) for approval  
- Receive funds after community verifies milestones  

**For Donors**  
- Contribute STX to a campaign  
- Track contributions and request refunds if needed  

**For Verifiers**  
- Vote on milestone approvals or flag suspicious campaigns  
- Verify fund usage and campaign details on-chain
